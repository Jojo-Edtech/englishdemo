# 英语备课组学情分析平台（演示原型）

在线预览：<https://jojo-edtech.github.io/englishdemo/>

面向高中英语备课组的学情分析 demo：导入成绩 → 批改诊断 → 错因定位 → 学情可视化 → 讲评课与家校报告，一条闭环走完。

## 主要功能

- **数据导入中心**：粘贴 Excel/CSV 本地解析（不上传服务器），字段映射与学生姓名匿名化
- **批改与作文工作台**：选择题批改结果、读后续写 rubric、DeepSeek 作文批改试用台
- **错因诊断**：单题错因、误选项 item analysis、学生分布
- **学情可视化**：题型×错因堆叠、能力雷达、热力图、知识图谱、重复错因追踪、**班级横向对比**
- **预警闭环**：学生预警中心，处理状态可流转（未处理 → 已安排 → 已复盘）
- **一键报告**：备课组 / 学生 / 家长三版报告，支持**语气切换（正式/简洁/鼓励）**、复制、下载 Markdown、打印存 PDF；家长版默认隐藏排名
- **讲评课备课单**：可复制、可导出教案大纲
- **学生端**：个人画像、进度曲线、任务闭环、**AI 跟进任务生成**、**个人错题本导出打印**

## 本地开发

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 产物在 dist/
```

推送到 `main` 分支后，GitHub Actions 自动构建并部署到 GitHub Pages。

## 设计依据

- [teacher-demand-audit.md](teacher-demand-audit.md)：基于小红书教师需求调研的功能优先级
- 参考主流学情分析产品（智学网、极课大数据、好分数）的报告分层、班级对比、错题本打印、个性化学习手册等模式

## 隐私边界

- 成绩数据只在浏览器本地解析，不发送到服务器
- DeepSeek Key 仅保存在本机浏览器，点击生成时才发送匿名摘要
- 家长版报告默认隐藏排名与班级比较
